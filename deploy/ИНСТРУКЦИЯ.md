```bash

# Mediasoup

# CORS
```

## 🔒 БЕЗОПАСНОСТЬ

### Firewall:

```bash
# Включение UFW
ufw --force enable

# Разрешение портов
ufw allow 22/tcp      # SSH
ufw allow 3000/tcp    # API
ufw allow 3001/tcp    # WebSocket
ufw allow 40000:40100/udp  # Mediasoup (голос/видео)
ufw allow 80/tcp      # HTTP
ufw allow 443/tcp     # HTTPS

# Проверка
ufw status
```

### HTTPS (рекомендуется):


## 🆘 ПРОБЛЕМЫ И РЕШЕНИЯ

### Ошибка: "Cannot connect to Docker daemon"

```bash
systemctl start docker
systemctl enable docker
```

### Ошибка: "Port already in use"

```bash
# Найти процесс
netstat -tulpn | grep :3000

# Убить процесс
kill -9 <PID>
```

### Ошибка Prisma:

```bash
cd /opt/VoiceFlow/server
npx prisma generate
npx prisma migrate reset
```

### Сервер не запускается:

```bash
# Проверка логов
cd /opt/VoiceFlow/server
npm run start:prod 2>&1 | tee server.log

# Проверка .env
cat .env
```

### Клиент не подключается:

1. Проверьте firewall на сервере:
   ```bash
   ufw status
   ```

2. Проверьте, что сервер доступен:
   ```bash
   curl http://89.169.4.132:3000/api/health
   ```

3. Проверьте `client\.env`:
   ```env
   VITE_API_URL=http://89.169.4.132:3000/api
   VITE_WS_URL=ws://89.169.4.132:3001
   ```
