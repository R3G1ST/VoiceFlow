                         🚀 БЫСТРЫЙ СТАРТ
================================================================================

1️⃣  ЗАПУСК ИНФРАСТРУКТУРЫ (Docker):
    
    cd C:\Users\R3G1S\VoiceFlow\deploy
    docker-compose up -d

2️⃣  НАСТРОЙКА СЕРВЕРА:
    
    cd C:\Users\R3G1S\VoiceFlow\server
    copy .env.example .env
    npm install
    npx prisma generate
    npm run start:dev

3️⃣  НАСТРОЙКА КЛИЕНТА (в другом окне):
    
    cd C:\Users\R3G1S\VoiceFlow\client
    npm install
    npm run dev

4️⃣  ОТКРЫТЬ БРАУЗЕР:
    
    http://localhost:5173

================================================================================
                         📊 ХАРАКТЕРИСТИКИ
================================================================================

✅ Звук:
   - Кодек: Opus
   - Битрейт: 64 kbps (как TeamSpeak)
   - Частота: 48 kHz
   - Каналы: 2 (стерео)
   - Задержка: < 50ms

✅ Видео:
   - Кодек: VP8/H264
   - Качество: 720p60 FPS
   - Демонстрация экрана: до 60 FPS

✅ Голосовые каналы:
   - Максимум: 50 человек
   - Постоянные каналы
   - Временные каналы

                         🔧 ТЕХНОЛОГИИ

Сервер:
- Node.js 20 + TypeScript
- NestJS - фреймворк
- Mediasoup - SFU для голоса/видео
- Prisma - ORM
- PostgreSQL - база данных
- Redis - кэш, pub/sub
- Socket.io - WebSocket

Клиент:
- Electron - десктоп
- React 18 + TypeScript
- Vite - сборка
- TailwindCSS - стили
- Zustand - state
- Simple-peer - WebRTC
