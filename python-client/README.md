# 🐍 VoiceFlow Python Client

Python десктоп клиент для VoiceFlow с использованием CustomTkinter.

---

## 🚀 БЫСТРЫЙ СТАРТ

### 1. Установка Python

Скачайте и установите Python 3.8+:
```
https://www.python.org/downloads/
```

### 2. Установка зависимостей

```bash
cd python-client
pip install -r requirements.txt
```

### 3. Запуск

```bash
python main.py
```

---

## 📦 СБОРКА В .EXE

### Windows

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "VoiceFlow" --icon="assets/icon.ico" main.py
```

**Готовый файл:** `dist/VoiceFlow.exe`

### Все платформы

```bash
pyinstaller --onefile --windowed --name "VoiceFlow" main.py
```

## 🐛 ОТЛАДКА
